### Hyprland
Hyprland конфиг разделён на конфиги поменьше для большей структуризации и простоты.

Также в этих файлах есть особая стилизация. Заголовки записывается так:
```conf
###
### Title
###

##
## Smaller title
##

# just comment
```

<!-- кто такие длинные ссылки делает :angry: :angry: -->
Подобно 
[callout-ам из GitHub Markdown](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax#alerts)
в конфиге есть такие же комментарии:

```conf
# -- Link -------------------------------------+
# https://wiki.hypr.land/Configuring/Keywords/ |
# ---------------------------------------------+

# (!) Note -------------+
# Please note something |
# ----------------------+
```

### Ignis

##### Структура
```
.config/ignis
├── config.py           # Основный файл
├── options.py          # Настройки [TODO]
├── style.scss          # Стили
└── widgets             # Виджеты
    ├── controls/       # Центр управления *aka* Control Center
    ├── clock.py        # Виджет часов
    ├── ...
    └── workspaces.py   # Виджет workspace-ов
```

##### Базовые команды
- **Запускайте `ignis init` прямо из консоли**, это полезно для просмотра логов
- `ignis inspector` для **Gtk инспектора**

## Требования к языку
Для всего кода/конфигурации действуют следующие правила:

- **Весь код/конфигурация должна быть на английском**
- **Весь текст, видимый пользователю, должен быть на английском**

## Документация
К документации относится информационные `.md` файлы находящиеся в этом репозитории.

- **Документация доступна на двух языках: английский (приориетный) и русский**
- **При своих изменениях проверяйте документацию, чтобы она оставалась достоверной**
- **Переводите на другие языки**
