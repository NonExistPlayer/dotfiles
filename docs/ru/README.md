# Требования

## Используемый софт
Это список приложений из самих дотфайлов, **не все из них обязательны в установке**, но
они являются частью оболочки. Рекомендуемые приложения выделены **жирным** шрифтом.

- **Hyprland**<br>
  Тайлинговый Wayland-композитор.
- **Ignis**<br>
  Фреймворк для создания десткопных оболочек.
- Kitty<br>
  Крутой эмулятор терминала.

## Требуемый софт
Это небольшой список из софта/библиотек. Необходимые программы выделены **жирным** шрифтом.

- **Git**<br>
  Система контроля версий. **Необходимо для установки**.<br>
  *Однако вы можете напрямую скачать дотфайлы с GitHub, хотя это бессмысленно.*
- **GNU Stow**<br>
  Организация всей важной конфигурации (пользователя) в один каталог (зачастую `~/.dotfiles`).<br>
  Автоматически создаёт ссылки из этого каталога в домашнюю директорию.
  **Необходимо для установки**.<br>
  *Как и в случае с Git, вы можете напрямую создать символические ссылки.*

Также рекомендуется ознакомиться с [требованиями Ignis](./.config/ignis/README.ru.md#Использование)
(если вы планируете использовать это).

# Установка
> [!WARNING]
> Установка только ручная, автоматической **не планируется**!
> Эти дотфайлы, созданы только для собственного использования
> и для тех, кто имеет опыт в настройке своего Linux дистрибутива.

1. Склонируйте git репозиторий в `~/.dotfiles` (можете выбрать любое имя)
```
git clone https://github.com/nonexistplayer/dotfiles ~/.dotfiles
```

2. Зайдите в `~/.dotfiles` и создайте ссылки с помощью `stow`
```
cd ~/.dotfiles
stow .
```
Это создаст символические ссылки в `~/.config`.

3. Перезагрузите систему или (пере)запустите Hyprland.

4. Наслаждайтесь.
